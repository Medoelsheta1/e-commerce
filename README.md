## E-commerce 
built with [react , redux-toolkit , Html , Css , Js , Es6 , bootstrap , react-router-dom , axios , react-icons , redux-persist , react-toastify]
using strapi as a backend server
### Advantages:
1- create new acc.\
2- login.\
3- add or remove items to shopping cart.\
4- filtering products.\
5- checkout process.\
6- sending order to my email.\
7- a profile for each user.\
8- search for any product.
